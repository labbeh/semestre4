- ajaxGET.html + ajax_info.txt : exemple de requete AJAX vers un fichier texte en utilisant la methode GET
- ajaxGETEtats.html + ajax_info.txt : meme chose que precedemment mais ici on trace tous les etats de 
                                      la requete au cours de son execution
- ajaxGETParam.html + ajaxGETParam.php : requete AJAX vers un script PHP cote serveur avec transmission de parametres dans la requete (methode GET)
- ajaxPOSTParam.html + ajaxPOSTParam.php : meme chose que precedemment mais avec la methode POST
